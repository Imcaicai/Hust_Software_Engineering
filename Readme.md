# 3 关于项目
### 💡项目简介
基于Android客户端的图书馆应用系统，主要功能如下：

- 登录、注册、填写个人信息
- 扫码借/还书、搜索图书、收藏图书、推荐图书
- 查看我的借阅、我的收藏、个人读书报告
- 专注打卡计时、赚取金币解锁城市建筑
- 查看图书馆最新活动、重要通知
- 查看图书馆每个楼层地图
- 查看读者指南、失物招领信息
